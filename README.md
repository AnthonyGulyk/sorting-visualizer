# Sorting Visualizer

Demo it live here: https://anthonygulyk.github.io/sorting-visualizer/

The sorting visualizer was built using React and Typescript.

The algorithims that are visualized are:
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
